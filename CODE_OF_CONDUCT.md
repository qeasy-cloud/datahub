# Code of Conduct

我们希望 QEasy DataHub 成为一个专业、友善、可协作的开源社区。

## 我们鼓励

- 尊重不同背景与经验的贡献者
- 提供具体、建设性的反馈
- 关注问题本身，而非针对个人
- 在讨论中保持事实、克制与礼貌

## 不被接受的行为

- 人身攻击、侮辱、骚扰或歧视性言论
- 恶意刷屏、挑衅、泄露他人隐私
- 明知有害仍持续破坏协作氛围

## 处理方式

维护者有权删除不当内容、关闭相关讨论，必要时限制参与权限。

## 联系方式

如需反馈社区行为问题，请通过 [轻易云官网联系页面](https://www.qeasy.cloud/contact) 与维护团队联系。
