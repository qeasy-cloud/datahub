# 更新日志

本文档记录轻易云 iPaaS 平台的版本发布历史、功能更新、优化改进及问题修复。通过语义化版本号（Semantic Versioning）管理，帮助用户快速了解各版本变更内容及升级建议。

> [!NOTE]
> 轻易云 iPaaS 采用语义化版本规范（SemVer），版本号格式为 **Major.Minor.Patch**：
> - **Major（主版本）**：不兼容的 API 变更或重大架构调整
> - **Minor（次版本）**：向下兼容的新功能添加
> - **Patch（修订版本）**：向下兼容的问题修复

## v3.5.1（2026-03-13）

> [!TIP]
> 当前最新版本，建议所有用户升级至此版本以获得最佳体验。

### 新增功能

- 🆕 **智能数据映射建议**：基于 AI 算法自动推荐字段映射关系，减少配置时间
- 🆕 **钉钉审批状态实时同步**：支持监听钉钉审批实例状态变更并自动触发集成流程
- 🆕 **OpenAPI 文档自动生成**：根据已配置的集成方案自动生成标准 OpenAPI 规范文档
- 🆕 **平台运维指南**：新增完整的平台运维手册，涵盖监控、备份、故障排查等内容

### 功能优化

- ⚡ **连接器加载性能优化**：大规模连接器列表加载速度提升 50%
- ⚡ **数据透视页面增强**：支持更多筛选条件和自定义列显示
- ⚡ **任务调度稳定性**：优化调度引擎，减少任务调度延迟

### 问题修复

- 🐛 修复金蝶云星空连接器在特定场景下的分页查询异常
- 🐛 修复数据映射页面字段类型识别错误的问题
- 🐛 修复定时任务跨天执行时的时间计算偏差

## v1.2.1（2022-08-01）

> [!IMPORTANT]
> 此为归档版本记录，展示早期产品迭代历程。当前版本已包含以下所有功能的基础形态及多次迭代改进。

### 新增功能

#### 数据可视化与治理

| 功能 | 描述 |
|------|------|
| **动态感知大屏** | 全新可视化大屏，一览所有集成数据运行状况，实时监控数据流向与异常 |
| **数据透视页面** | 支持自定义数据表格，通过透视表可视化运维数据，提供多维度数据分析能力 |
| **数据建模功能** | 新增集成数据建模能力，对集成数据进行规范化治理，提升数据质量 |

#### 智能运维与告警

| 功能 | 描述 |
|------|------|
| **钉钉运维机器人小助理** | 自动感知异常与集成状况，推送告警至钉钉群；支持在钉钉群内与机器人交互查询 |
| **异常数据总览优化** | 更直观的异常数据展示界面，支持一键导出异常记录 |

#### 调度与编排

| 功能 | 描述 |
|------|------|
| **调度前置条件配置** | 支持编排多个关联集成方案的运行逻辑，实现复杂业务流程自动化 |
| **多调度者模式** | 允许为不同集成方案分配多个调度者，提升调度灵活性与资源利用率 |
| **可视化定时策略配置** | 通过可视化界面配置调度者运行时间，无需手工编写 Linux Crontab |

#### 数据操作增强

| 功能 | 描述 |
|------|------|
| **自定义过滤条件可视化配置** | 自定义过滤条件无需编写元数据代码，通过界面可视化配置 |
| **数据透视多参数过滤** | 数据透视页面支持可视化的多参数数据筛选 |
| **数据透视批量操作** | 支持对数据进行批量操作，包括批改状态、字段修改、批量删除 |
| **数据透视枚举粘贴搜索** | 支持通过枚举值粘贴方式进行批量搜索 |

#### 函数与关联助手

| 功能 | 描述 |
|------|------|
| **高级函数助手 `_function`** | 通过可视化界面配置函数，无需编写代码 |
| **关联数据助手 `_findCollection`** | 关联数据查询通过界面配置完成，无需编写代码 |

#### API 开放能力

| 功能 | 描述 |
|------|------|
| **API 网关服务** | 集成平台数据支持更开放的接口能力，详见[对外开放接口文档](../developer/open-data-api) |

### 功能优化

- **集成方案业务分组**：支持选择现有分组或手工新增分组，提升方案管理效率
- **数据全生命周期链路**：可通过源队列、源数据观察整个数据链路过程，增强可观测性
- **进程管理功能优化**：改进进程管理界面与操作体验

## 版本支持策略

| 版本类型 | 支持周期 | 更新内容 |
|----------|----------|----------|
| **Major** | 24 个月 | 重大版本，提供长期支持（LTS） |
| **Minor** | 6 个月 | 功能版本，持续更新与修复 |
| **Patch** | 持续 | 问题修复，建议及时升级 |

> [!WARNING]
> 已结束支持的版本不再提供安全更新与技术支持，建议用户及时升级至最新稳定版本。

## 升级指南

### 升级前准备

1. **备份数据**：升级前请完整备份配置数据与运行日志
2. **查看兼容性说明**：阅读目标版本的兼容性变更说明
3. **测试环境验证**：建议在测试环境先行验证升级流程

### 升级方式

| 部署方式 | 升级方法 |
|----------|----------|
| SaaS 版本 | 自动升级，无需操作 |
