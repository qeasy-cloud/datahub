# 纷享销客连接器

本文档介绍轻易云 iPaaS 与纷享销客 CRM 平台的集成配置方法。

## 平台简介

纷享销客是国产领先的移动 CRM 平台，提供销售自动化、营销管理、客户服务等功能，支持多终端访问。轻易云 iPaaS 提供纷享销客连接器，支持与 ERP、OA 等系统的数据集成。

## 连接配置

### 前置条件

- 纷享销客企业版账号
- 开通 API 接口权限
- 获取 AppKey 和 AppSecret

### 配置步骤

1. 登录纷享销客管理后台
2. 进入 **设置 → 开放平台 → API 管理**
3. 创建应用并获取 AppKey 和 AppSecret
4. 在轻易云控制台创建连接器

## 集成方案配置

### 常用接口

| 接口 | 说明 |
|------|------|
| 查询对象数据 | 查询客户、联系人、商机等 |
| 创建对象数据 | 写入客户、订单等 |
| 更新对象数据 | 更新记录信息 |
| 删除对象数据 | 删除指定记录 |

### 数据对象

纷享销客的核心数据对象：
- Account（客户）
- Contact（联系人）
- Opportunity（商机）
- Order（订单）
- Product（产品）
