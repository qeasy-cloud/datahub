## 账号相关

### 如何注册账号？

访问 [轻易云官网](https://www.qeasy.cloud)，点击「免费注册」，支持手机号、邮箱、钉钉、企业微信、飞书等多种注册方式。

### 忘记密码怎么办？

在登录页面点击「忘记密码」，通过手机号或邮箱验证后重置密码。

### 如何添加团队成员？

进入「系统设置」→「成员管理」，输入成员手机号或邮箱发送邀请。

## 连接器相关

### 连接失败如何排查？

1. 检查网络连通性（ping 或 telnet 测试）
2. 验证账号密码或 API Key 是否正确
3. 检查目标系统是否开启了 IP 白名单
4. 确认账号具有足够的权限
5. 查看详细错误信息进行针对性排查

### 支持哪些系统？

轻易云 iPaaS 已支持 500+ 系统，包括：
- ERP：金蝶、用友、SAP、Oracle
- OA：钉钉、飞书、企业微信、泛微
- 电商：旺店通、聚水潭、管易云
- 数据库：MySQL、Oracle、SQL Server、MongoDB

完整列表请参考 [已接入平台](./introduction/connected-platforms)。

## 方案相关

### 如何调试方案？

1. 使用「调试模式」单步执行
2. 查看「执行日志」了解详细过程
3. 使用「数据预览」查看中间结果
4. 设置「断点」暂停执行检查状态

### 任务失败如何处理？

1. 查看执行日志定位错误
2. 根据错误类型进行处理：
   - 连接错误：检查网络或认证
   - 数据错误：检查数据映射和转换
   - 系统错误：联系技术支持
3. 修复问题后重新执行

## 数据相关

### 数据映射错误怎么办？

1. 检查字段名称是否正确
2. 确认数据类型匹配
3. 使用「值格式化器」进行转换
4. 使用「调试」查看实际数据

### 如何确保数据不重复？

1. 配置幂等键（唯一标识字段）
2. 使用「先查后写」模式
3. 在目标系统设置唯一索引
4. 使用「数据版本」控制

## 性能相关

### 同步速度太慢怎么办？

优化建议：
1. 增加批量处理大小
2. 使用并发执行
3. 优化数据查询条件
4. 使用增量同步替代全量
5. 升级实例规格

### 大数据量如何处理？

1. 使用分页查询
2. 分批处理写入
3. 配置合理的超时时间
4. 使用异步处理模式
5. 考虑使用 CDC 实时同步

## 计费相关

### 如何查看账单？

进入「费用中心」→「账单管理」，可查看详细的使用量和费用明细。

### 免费额度是多少？

新用户注册即送 ¥100 代金券，可用于抵扣首月费用。免费试用版提供 14 天完整功能体验。

### 如何升级套餐？

在「费用中心」选择需要的套餐，在线支付后即可升级，即时生效。
