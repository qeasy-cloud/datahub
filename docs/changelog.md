# 更新日志

本文档记录轻易云 iPaaS 的版本更新历史。

## 版本说明

轻易云 iPaaS 版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：

- 主版本号：不兼容的 API 修改
- 次版本号：向下兼容的功能新增
- 修订号：向下兼容的问题修复
