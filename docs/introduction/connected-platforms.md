### 接入申请

如需申请新的平台接入，请提供以下信息：

1. 平台名称和官网链接
2. 开放平台文档或 API 文档
3. 您的使用场景描述
4. 预计使用量

我们将在收到申请后的 5 个工作日内与您联系，评估接入可行性。
