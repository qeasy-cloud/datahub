# 环境配置

本文档介绍使用轻易云 iPaaS 前的环境准备工作，包括浏览器要求、网络配置等。

## 浏览器要求

为了获得最佳的使用体验，请使用以下浏览器访问轻易云 iPaaS：

| 浏览器 | 最低版本 | 推荐版本 | 说明 |
|-------|---------|---------|------|
| Google Chrome | 90+ | 最新版 | 推荐 |
| Microsoft Edge | 90+ | 最新版 | 推荐 |
| Firefox | 88+ | 最新版 | 支持 |
| Safari | 14+ | 最新版 | 支持 |
| 360 极速浏览器 | 13+ | 最新版 | 支持 |

> [!WARNING]
> 请避免使用 IE 浏览器访问，部分功能可能无法正常使用。

### 浏览器设置

#### 启用 JavaScript

轻易云 iPaaS 需要 JavaScript 支持，请确保浏览器未禁用 JavaScript。

**Chrome 设置步骤：**

1. 打开 Chrome 设置
2. 进入「隐私和安全」→「网站设置」
3. 找到「JavaScript」，选择「允许」

#### 允许 Cookie

请确保浏览器允许使用 Cookie，以便正常登录和保存用户偏好设置。

## 网络配置

### 访问域名

轻易云 iPaaS 使用以下域名，请确保网络可以正常访问：

| 域名 | 用途 | 协议 |
|-----|------|------|
| www.qeasy.cloud | 主站 | HTTPS |
| api.qeasy.cloud | API 服务 | HTTPS |
| console.qeasy.cloud | 控制台 | HTTPS |
| cdn.qeasy.cloud | 静态资源 | HTTPS |

### 防火墙配置

如果您的网络环境有防火墙限制，请确保以下端口开放：

| 端口 | 协议 | 用途 |
|-----|------|------|
| 443 | TCP | HTTPS 访问 |
| 80 | TCP | HTTP 重定向 |

### 代理设置

如果您使用代理服务器访问互联网，请确保：

1. 代理服务器支持 WebSocket 协议
2. 代理服务器未拦截上述域名
3. SSL 证书校验正常

## 私有化部署环境要求

如果您使用私有化部署版本，需要准备以下环境：

### 服务器要求

| 部署规模 | CPU | 内存 | 磁盘 | 说明 |
|---------|-----|------|------|------|
| 小型 | 4 核 | 8 GB | 100 GB | 适合 POC 验证 |
| 中型 | 8 核 | 16 GB | 500 GB | 适合中小型企业 |
| 大型 | 16 核+ | 32 GB+ | 1 TB+ | 适合大型企业 |

### 软件依赖

#### 必需组件

| 组件 | 版本要求 | 说明 |
|-----|---------|------|
| Docker | 20.10+ | 容器运行环境 |
| Docker Compose | 2.0+ | 容器编排 |
| Linux 内核 | 3.10+ | 操作系统 |

#### 支持的操作系统

- CentOS 7.x / 8.x
- Ubuntu 18.04 LTS / 20.04 LTS / 22.04 LTS
- Red Hat Enterprise Linux 7.x / 8.x
- Debian 10 / 11

### 数据库要求

私有化部署需要准备以下数据库：

| 数据库 | 版本 | 用途 |
|-------|------|------|
| MySQL | 5.7+ / 8.0+ | 业务数据存储 |
| Redis | 6.0+ | 缓存和消息队列 |
| MongoDB | 4.4+ | 日志和大数据存储 |

### 网络端口

私有化部署需要开放以下端口：

| 端口 | 用途 | 访问来源 |
|-----|------|---------|
| 80 | HTTP 入口 | 所有用户 |
| 443 | HTTPS 入口 | 所有用户 |
| 3306 | MySQL | 应用服务器 |
| 6379 | Redis | 应用服务器 |
| 27017 | MongoDB | 应用服务器 |

## 开发环境配置

如果您需要进行二次开发或自定义连接器开发，需要准备以下环境：

### Python 开发环境

```bash
# 安装 Python 3.8+
python --version

# 安装依赖包
pip install requests
pip install pandas
```

### Java 开发环境

```bash
# 安装 JDK 11+
java -version

# 安装 Maven
mvn -version
```

### Node.js 开发环境

```bash
# 安装 Node.js 16+
node --version
npm --version
```

## 验证环境

完成环境配置后，可以通过以下方式验证：

### 浏览器验证

1. 访问 [轻易云官网](https://www.qeasy.cloud)
2. 检查页面是否正常加载
3. 尝试登录控制台，确认功能正常

### 网络验证

```bash
# 测试网络连通性
ping www.qeasy.cloud

# 测试 HTTPS 访问
curl -I https://www.qeasy.cloud
```

### 私有化部署验证

```bash
# 检查 Docker 状态
docker version
docker-compose version

# 检查端口占用
netstat -tlnp | grep -E '80|443|3306|6379'
```

## 故障排查

### 无法访问控制台

| 现象 | 可能原因 | 解决方法 |
|-----|---------|---------|
| 页面空白 | JavaScript 被禁用 | 启用 JavaScript |
| 加载缓慢 | 网络不稳定 | 检查网络连接 |
| 证书错误 | 系统时间不正确 | 校准系统时间 |
| 白屏 | 浏览器不兼容 | 更换推荐浏览器 |

### 私有化部署问题

| 现象 | 可能原因 | 解决方法 |
|-----|---------|---------|
| 容器启动失败 | 端口被占用 | 检查并释放端口 |
| 数据库连接失败 | 配置错误 | 检查数据库配置 |
| 服务不可用 | 依赖服务未启动 | 按顺序启动服务 |
