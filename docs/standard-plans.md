# 标准集成方案

轻易云 iPaaS 提供开箱即用的标准集成方案，帮助企业快速实现常见业务场景的集成。

## 方案分类

```mermaid
mindmap
  root((标准方案))
    电商方案
      跨境电商
      国内电商
      电商财务一体化
    协同方案
      OA 审批集成
      费用报销
      考勤同步
    数据方案
      数据中台
      报表同步
      主数据管理
    行业方案
      MES 集成
      WMS 对接
      CRM 集成
```
